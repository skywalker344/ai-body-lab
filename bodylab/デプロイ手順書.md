# AI BODY LAB - Vercelデプロイ手順書

## 所要時間：約15〜20分
## 必要なもの：GitHubアカウント（無料）、Vercelアカウント（無料）

---

## STEP 1：GitHubにリポジトリを作る（5分）

1. https://github.com にアクセスしてログイン（アカウントがなければ無料登録）
2. 右上の「+」→「New repository」をクリック
3. 設定：
   - Repository name: `ai-body-lab`
   - Public（公開）を選択
   - 「Create repository」をクリック

---

## STEP 2：ファイルをGitHubにアップロード（5分）

1. 作成したリポジトリページで「uploading an existing file」をクリック
2. このフォルダ内のファイルを**フォルダ構造を保ったまま**アップロード：

```
ai-body-lab/
  ├── package.json
  ├── vite.config.js
  ├── index.html
  ├── tutorial.html
  └── src/
      ├── main.jsx
      └── App.jsx
```

**注意：** `src` フォルダは「Add file」→「Upload files」では
フォルダごとドラッグ＆ドロップできます。

3. 「Commit changes」をクリック

---

## STEP 3：Vercelでデプロイ（5分）

1. https://vercel.com にアクセス
2. 「Sign Up」→「Continue with GitHub」でGitHubアカウントと連携
3. ダッシュボードで「Add New Project」をクリック
4. `ai-body-lab` リポジトリを選択して「Import」
5. 設定（変更不要、デフォルトのままでOK）：
   - Framework Preset: Vite（自動検出される）
   - Build Command: `npm run build`
   - Output Directory: `dist`
6. 「Deploy」をクリック
7. 1〜2分でデプロイ完了 🎉

---

## STEP 4：URLを確認する

デプロイ完了後、以下のようなURLが発行されます：
```
https://ai-body-lab.vercel.app
```
または
```
https://ai-body-lab-yourname.vercel.app
```

---

## STEP 5：チュートリアルページのURLを更新する

`tutorial.html` の中の `APP_URL_HERE` を実際のURLに置き換える：

1. GitHubの `tutorial.html` を開いて編集（鉛筆アイコン）
2. `APP_URL_HERE` を `https://ai-body-lab.vercel.app` に3箇所置き換え
3. 「Commit changes」→ Vercelが自動で再デプロイ

チュートリアルページのURLは：
```
https://ai-body-lab.vercel.app/tutorial.html
```

## 独自ドメインを使う場合（任意・年1,500円程度）

1. Namecheap / お名前.com などでドメイン取得
   - 例：`ai-bodylab.jp`（.jpは年1,500円前後）
2. Vercelダッシュボード → プロジェクト → 「Settings」→「Domains」
3. 取得したドメインを入力 → 指示に従いDNS設定
4. 設定後15分〜1時間で反映

---

## よくあるトラブル

### ビルドエラーが出る場合
→ `package.json` と `vite.config.js` が正しい場所にあるか確認

### 「Page not found」になる場合
→ Vercelの「Settings」→「Build & Development Settings」で
  Framework PresetがViteになっているか確認

### GitHubにsrcフォルダがアップロードできない場合
→ GitHub Desktopアプリ（無料）を使うと
  フォルダ構造ごと簡単にアップロードできます
  https://desktop.github.com

---

## 更新したいとき

アプリを修正した場合：
1. GitHubのファイルを編集して「Commit changes」
2. Vercelが自動で再デプロイ（約1〜2分）
